# ovpn script
Openvpn server installer

*tested on Ubuntu 14.0.4.5

# Features
```
UDP 
port 443
AES-256-CBC
SHA512
TLS-DHE-RSA-WITH-AES-128-GCM-SHA256
```




# Usage
```
wget https://raw.githubusercontent.com/xumr0x/ovpn/master/openvpn-install.sh
chmod +x openvpn-install.sh
./openvpn-install.sh
```
